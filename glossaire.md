# Glossaire

## Consignes

L’exercice consiste à produire une définition d’un terme relatif au domaine de la documentation muséale. Vous devez produire un texte concis mais rigoureux et précis destiné à des spécialistes de la muséologie. L’entrée pourra être complétée d’une courte sélection de références bibliographiques ou webographiques.

- Créez un compte sur la plateforme GitHub pour pouvoir modifier ce document.
- Indiquez votre nom sous l’entrée que vous avez choisie en modifiant le document dans l’interface.
- Après avoir rédigé le texte de votre entrée sur votre ordinateur,
insérez votre travail dans le document, dans le message de modification précisez votre nom si votre pseudo n’est pas explicite sur la plateforme.
